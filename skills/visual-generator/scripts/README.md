# visual-generator / scripts

This skill emits `visual.json`, `page.json`, `pages.json`, `report.json`, `version.json`, and `definition.pbir` files. The supporting templates and scripts live alongside `query-to-pbip`:

| Resource | Location |
|---|---|
| Visual templates (one per `visualType`) | `../../query-to-pbip/assets/visual-templates/` |
| PBIP scaffolding script | `../../query-to-pbip/scripts/scaffold_pbip.py` |

To produce a single visual.json by hand, follow the steps in `SKILL.md`. To produce a complete PBIP, invoke `query-to-pbip`.
